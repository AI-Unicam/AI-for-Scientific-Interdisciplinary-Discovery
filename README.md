
# AI for Scientific & Interdisciplinary Discovery

AI methods for accelerating discovery across science, engineering, medicine, social sciences, and the humanities, including scientific modelling, simulation, hypothesis generation, and domain-specific AI systems.

## Contributing (University of Camerino collaborators)

Researchers working on this topic are welcome to share their work through this repository.

To keep projects independent while making them discoverable, please add your repository as a Git submodule.

```bash
git submodule add https://github.com/<username>/<repository>.git <repository>
git commit -m "Add <repository> as submodule"
git push
```

Please place the submodule in the repository root and open a Pull Request with a short description of the project.
